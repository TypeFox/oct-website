Open Collaboration Tools

Documentation Playground Support

Collaborative Editing for your IDE or custom Editor

Open Collaboration Tools is an set of open-source tools, libraries and extensions to allow real-time collaborative editing across devices and tools

It's built to be easily extensible, to enable creating custom clients or modifying the server to suit your specific needs

Features

Public Server

Extend and deploy your own server

Multiple already available clients

Clients

VS Code - OCT Tools
Eclipse Theia - @theia/collaboration
Monaco
